# 06: Advance Click Point 選択 UI（透明オーバーレイ）

## Parent

`.scratch/mvp/PRD.md`

## What to build

01 でハードコードされていた Advance Click Point を、視覚的に画面上から1点クリックで指定できるようにする。

- フロント：「クリック位置選択」ボタン押下 → 全画面を覆う半透明オーバーレイ。
  - クリックで確定 → オーバーレイ消滅 → バーに「クリック位置指定済み」状態表示
  - Esc キーでキャンセル
- クリック座標（グローバルスクリーン座標）を Go 側に渡し、`app.go` 内で保持。CaptureSession 開始時に AdvanceClickPoint として渡す。
- 再度「クリック位置選択」ボタンを押せば上書き可能。
- 05 のオーバーレイ実装と機構を共有可能なら共通化する（必須ではない）。

## Acceptance criteria

- [ ] 「クリック位置選択」ボタンを押すと全画面に半透明オーバーレイが出る
- [ ] クリックで確定し、バー上に「クリック位置指定済み」表示が出る
- [ ] Esc でキャンセルすると元の状態に戻る
- [ ] 確定した座標で開始すると、各ステップでその座標がクリックされる
- [ ] クリック位置未指定の状態では開始ボタンが押せない
- [ ] 再選択で上書きできる
