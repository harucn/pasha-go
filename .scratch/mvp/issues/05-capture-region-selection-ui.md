# 05: Capture Region 選択 UI（透明オーバーレイ）

## Parent

`.scratch/mvp/PRD.md`

## What to build

01 でハードコードされていた Capture Region を、視覚的に画面上からドラッグ選択できるようにする。

- フロント：「範囲選択」ボタン押下 → 全画面を覆う半透明オーバーレイ（フロント側で全画面 div として実装、または Wails の 2nd ウィンドウとして実装）。
  - マウスドラッグで矩形ハイライト表示
  - マウスアップで確定 → オーバーレイ消滅 → バーに「範囲指定済み」状態表示
  - Esc キーでキャンセル → オーバーレイ消滅 → バーは未指定状態のまま
- 矩形のスクリーン座標を Go 側に渡し、`app.go` 内で保持。CaptureSession 開始時にこれを CaptureRegion として渡す。
- マウス座標は OS のグローバルスクリーン座標である必要があり、Wails の座標変換が必要なら `runtime` API を介して取得する。
- 再度「範囲選択」ボタンを押せば上書き可能。

## Acceptance criteria

- [ ] 「範囲選択」ボタンを押すと全画面に半透明オーバーレイが出る
- [ ] ドラッグ中に矩形がリアルタイムで描画される
- [ ] マウスアップで確定し、バー上に「範囲指定済み」表示が出る
- [ ] Esc でキャンセルすると元の状態に戻る
- [ ] 確定した範囲で開始すると、生成 PDF のページに指定範囲の画像が入っている（画面全体ではない）
- [ ] 範囲未指定の状態では開始ボタンが押せない
- [ ] 再選択で上書きできる
