# PDF を逐次追記方式で生成する

Capture Session の Output Document（マルチページPDF）は、各 Capture Step ごとに**ページ単位で PDF へ追記**する方式を採用する。

中間 PNG をディスクに保存しないため一時ファイルの汚染がない一方、完全インメモリで最後にまとめてPDF生成する案と異なり、途中でクラッシュしても**その時点までのページは部分PDFとして残る**。撮影は数十〜数百ステップに及ぶ長時間タスクになり得るため、クラッシュ耐性を優先した。

## Considered Options

- 完全インメモリ → 最後に PDF 出力：実装は最シンプルだが、途中失敗で全ロス
- 中間 PNG 保存 → 最後に PDF 化 → PNG 削除：クラッシュ耐性はあるが、一時ファイル管理が必要
- **PDF 逐次追記（採用）**：両者の良いとこ取り
